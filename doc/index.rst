Before system installation
==========================

Remember to install any upgrades prior to installing Linux. `BIOS upgrade <http://www.dell.com/support/drivers/us/en/19/DriverDetails/DriverFileFormats/Product/latitude-xt?DriverId=R254859&FileId=2731129404&urlProductCode=False>`_ and `N-Trig touchscreen firmware upgrade <http://www.ntrig.com/Content.aspx?Page=Downloads_Drivers>`_ can only be applied via Windows.
